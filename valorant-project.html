User Interface Design

Valorant agent activation screen

Client

Riot Inc. | Jk

Date

15 Apr 2021

Category

UI/UX

The process of developing the UI design for the Valorant agent activation page was extremely rewarding. To get inspired and understand the layout and visual elements I wanted to include in my design, I began by researching previous designs. From there, I created page sketches and refined the layout and elements until I was happy with the overall look.

While creating this UI design, I learned a lot. I learned more about the importance of user experience and how to design an engaging and intuitive user interface. I also learned about the technical aspects of user interface design, such as how to structure a page for maximum loading speed. In addition, I gained knowledge of color theory, typography, and other visual elements that are essential when creating a UI design.
