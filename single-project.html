Logo Design

For a discord server

Client

Anonymous #0222

Date

22 Feb 2021

Category

Illustration

It was a pleasure to design a logo for the Communitea Discord server. I wanted to make sure the logo was unique to the server and reflected the spirit of the community. I used a vibrant color palette and a simple yet elegant font to create an inviting and cheerful logo. The design also included a bulb, which symbolized the server's core concept of communal gathering and connection.

Overall, I'm pleased with the outcome, and I believe it perfectly conveys the core values of the Communitea server.
